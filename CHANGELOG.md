
# Changelog

## [1.2.8] - 2026-06-23

### Added
- Калибровка антенны по таблице поправок: накопление точек при движении вокруг маяка с последующим построением таблицы ошибок по секторам
- Возможность скачать полный отчёт калибровки (сырые данные + таблица) для анализа

### Changed
- Кнопка калибровки антенны перенесена из меню "Лог" в "Инструменты"
- Загрузка калибровочной таблицы и сброс вынесены в колбеки для устранения зависимости от глобальных переменных

### Fixed
- Исправлена загрузка калибровочной таблицы из файла в AntennaCorrector

## [1.2.7] - 2026-06-16

### Added 
- Экспорт трека антенны в формате GGA
- Предупреждение при закрытии страницы, если есть треки или журнал

### Fixed
- Время при экспорте в формате GGA


## [1.2.6] - 2026-06-10

### Fixed
- Отображение карточки маяка
- Автоцентрирование при нажатии на карточку маяка или станции

## [1.2.5] - 2026-06-10

### Changed
- Рефакторинг: вынесены модули `ui-topo.js`, `ui-calibration.js`, `ui-canvas.js`, создан `export.js`
- Устранено дублирование кода геодезических расчётов (использование `GeoUtils.geoToScreen()`)
- Оптимизирована инициализация и обработчики событий (мышь, тач)
- Упрощена работа с масштабом и панорамированием через `UICanvas`

### Fixed
- добавлена глубина при экспорте в формат KML


## [1.2.4] - 2026-06-08

### Added
- Инструмент "Линейка" для измерения расстояний на карте (две точки, резиновая линия, работает при зуме/панорамировании)

### Changed
- Рефакторинг: выделены модули `ui-themes.js`, `ui-settings.js`, `ui-ruler.js`
- Улучшены цвета сетки в тёмных темах (Jack Black, Dark Contrast, VAX)
- Версия приложения обновлена до 1.2.4


## [1.2.3] - 2026-06-08
### Added
- Настройка максимальной скорости маяка для ACHOD-фильтра (0.5 - 5 м/с)
- Разные режимы работы ACHOD-фильтра при разных дальностях
- Пропуск пустого времени при воспроизведении логов (до первой точки маяка)
- Ускоренное воспроизведение логов (2x, 4x, 8x)

### Changed
- ACHOD-фильтр (DH-filter) адаптирован для работы с разными размерами очередей
- Улучшено сохранение/загрузка настроек
- Добавлена настройка максимальной скорости маяка

### Fixed
- Исправлено некорректное отображение сохранённой максимальной скорости в окне настроек
- Ручная топопривязка теперь позволяет изменять только азимут без перезаписи координат


## [1.2.2] - 2026-06-06
### Added
- Отображение отвергнутых фильтром точек (серые крестики) для диагностики
- Температура с маяка в карточке (в строке с VCC)
- Адаптивный порог DH-фильтра в зависимости от дальности
- Темы Jack Black, Jack White, VAX добавлены в цикл переключения тем

### Changed
- Сглаживатель координат отключается на дистанциях > 1000 м
- Азимут в карточке: в скобках если измеренный, без скобок если абсолютный

### Fixed
- Дальность в карточке маяка всегда показывает актуальное измеренное значение


## [1.2.1] - 2026-06-05
### Added
- Улучшен анализ логов (расширенная статистика по heading)
- Корретный пересчет девиации и склонения в сообщении HDG

### Fixed
- Парсинг логов учитывает разные источники (GNSS, HDM)

## [1.2.0] - 2026-06-02
### Added
- Экспорт PSIMSSB с UTM координатами (C,E)
- Медианный фильтр координат
- Корректор угла антенны по калибровочной таблице
- Авто-расчёт скорости звука по UNESCO (температура + солёность)

### Fixed
- Время в GGA/PSIMSSB без даты (hhmmss)
- xM/yM в градусах вместо радианов

## [1.1.0] - 2026-05-15
### Added
- Экспорт PSIMSSB (C,H — Head-Up)
- Сохранение относительных координат в трек

## [1.0.0] - 2026-04-01
